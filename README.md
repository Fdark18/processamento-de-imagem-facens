# Processamento de Imagens - Facens

Este repositório contém ferramentas, scripts e datasets voltados para o estudo e aplicação de técnicas de Processamento Digital de Imagens.

## Estrutura do Projeto

### `exec01/` - Ferramentas de Redimensionamento Aula

### `exec02/` - Ferramentas de Redimensionamento exercicios

### `data/` - Datasets
Arquivos auxiliares contendo conjuntos de caracteres, úteis para tarefas de reconhecimento de texto ou testes de legibilidade.

- **alphabet_36.txt**: Caracteres alfanuméricos básicos (0-9, a-z).
- **alphabet_94.txt**: Conjunto ASCII estendido (inclui maiúsculas e símbolos).

## Instalação e Dependências

O projeto requer Python 3 e as bibliotecas de processamento numérico e de imagem.

```bash

# via uv (como observado no ambiente)
uv pip install opencv-contrib-python numpy
```
